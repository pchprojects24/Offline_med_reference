Airway Procedure

BVM + OPA/NPA

Indications

  • Inadequate breathing (hypoventilation), apnea, or impending respiratory failure
  • Need for assisted ventilation during resuscitation, sedation, transport, or airway management
  • Airway adjuncts:
    • OPA: unconscious/absent gag to prevent tongue obstruction
    • NPA: reduced consciousness or poor airway tone when OPA not tolerated (if not contraindicated)

Contraindications / cautions

  • BVM:
    • Caution with excessive volume/pressure (gastric insufflation, aspiration, barotrauma)
  • OPA:
    • Intact gag reflex / conscious patient (vomiting/laryngospasm risk)
  • NPA:
    • Suspected basilar skull fracture or significant midface trauma (avoid)
    • Significant epistaxis risk/coagulopathy/anticoagulation (use judgment)

Equipment

  • BVM with oxygen reservoir + O2 source (high-flow)
  • Appropriately sized mask (adult) + optional PEEP valve if used locally
  • OPA set (sizes) and/or NPA set + water-based lubricant
  • Suction (ready and working)
  • Monitoring: SpO2 + ECG; ETCO2 if available

Steps (checklist)

1) Prepare and position

  • Call for help if needed; assign airway/ventilation role
  • Position: head-tilt–chin-lift (no trauma) or jaw-thrust (trauma/unknown)
  • Suction secretions/vomit early; remove dentures only if loose

2) Airway adjunct (as indicated)

  • OPA sizing: corner of mouth to angle of mandible
    • Insert with appropriate technique; ensure it sits properly without pushing tongue back
  • NPA sizing: nostril to earlobe/angle of jaw (device-dependent)
    • Lubricate; insert gently; stop if significant resistance/bleeding

3) Mask seal and ventilation

  • Choose two-person technique when available (best seal + jaw support)
  • Maintain tight seal; lift mandible into mask (avoid pushing mask down hard)
  • Deliver slow breaths with visible (not excessive) chest rise
  • Rate: ventilate per scenario and local protocol (avoid hyperventilation)
  • If using PEEP locally: apply only if trained and monitor BP/oxygenation

4) Ongoing reassessment

  • Confirm effectiveness:
    • Visible chest rise, improving SpO2, improving skin color, improving mental status (if applicable)
    • ETCO2 waveform if available
    • Auscultate for bilateral breath sounds when feasible
  • If ineffective:
    • Reposition head/jaw, re-seat mask, switch to two-person, suction again
    • Add/adjust OPA/NPA
    • Escalate to SGA/ETT per scope and local protocol if needed

Confirmation / success criteria

  • Consistent chest rise + improved oxygenation/ventilation indicators
  • ETCO2 present and appropriate trend (if monitored)
  • Minimal leak; minimal gastric insufflation

Complications + immediate management

  • Gastric insufflation/vomiting/aspiration
    • Reduce volume/pressure; slow ventilation; suction; lateral positioning if feasible
  • Hypotension with excessive intrathoracic pressure/PEEP
    • Reduce ventilation pressure/PEEP; reassess perfusion
  • Mask leak / poor ventilation
    • Two-person seal; reposition; adjuncts; escalate airway

Aftercare / monitoring

  • Continue monitoring SpO2, RR, effort, mental status; ETCO2 if available
  • Reassess adjunct position frequently
  • Document response and any complications
  • Plan next step (continue BVM vs SGA/ETT) based on trajectory and context
